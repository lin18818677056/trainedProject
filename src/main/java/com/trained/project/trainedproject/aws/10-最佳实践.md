## 10.1 成本优化全面指南

### 10.1.1 AWS 定价模型详解

#### 按需付费 (On-Demand)
```
特点:
  ✓ 无前期费用
  ✓ 按秒/小时计费
  ✓ 随时停止
  ✓ 单价最高
  
适用场景:
  - 短期不稳定负载
  - 开发和测试
  - 不可预测的使用模式
```

#### 预留实例 (Reserved Instances)
```
承诺期限:
  - 1 年期：节省约 30-40%
  - 3 年期：节省约 50-60%
  
付款选项:
  - 全部预付：最大折扣
  - 部分预付：中等折扣
  - 无预付：较小折扣
  
适用场景:
  - 稳定的生产负载
  - 长期运行的应用
  - 数据库服务
```

#### Spot 实例
```
折扣：可达 90%
风险：可能被中断（提前 2 分钟通知）
  
适用场景:
  - 批处理任务
  - 数据分析
  - 容错应用
  - CI/CD流水线
  
不适用:
  - 关键业务
  - 需要持久运行的服务
  - 数据库主节点
```

#### Savings Plans
```
承诺每小时消费金额
自动应用于符合的资源
比 RI 更灵活
  
类型:
  - Compute Savings Plans: EC2, Lambda, Fargate
  - EC2 Instance Savings Plans: 特定实例家族
  
节省：可达 72%
```

### 10.2.3 自动化安全脚本

#### 检查未使用的安全组
```python
import boto3

ec2 = boto3.client('ec2')

def check_unused_security_groups():
    # 获取所有安全组
    response = ec2.describe_security_groups()
    
    # 获取正在使用的安全组
    instances = ec2.describe_instances()
    used_sg_ids = set()
    
    for reservation in instances['Reservations']:
        for instance in reservation['Instances']:
            for sg in instance['SecurityGroups']:
                used_sg_ids.add(sg['GroupId'])
    
    # 找出未使用的
    for sg in response['SecurityGroups']:
        if sg['GroupId'] not in used_sg_ids and not sg['GroupName'].startswith('default'):
            print(f"Unused SG: {sg['GroupId']} - {sg['GroupName']}")
```

### 10.4.3 自动化运维脚本

#### 自动备份脚本
```python
import boto3
from datetime import datetime, timedelta

rds = boto3.client('rds')
ec2 = boto3.client('ec2')

def create_rds_snapshot():
    """创建 RDS 快照"""
    db_identifier = 'my-production-db'
    snapshot_name = f"{db_identifier}-{datetime.now().strftime('%Y%m%d')}"
    
    rds.create_db_snapshot(
        DBSnapshotIdentifier=snapshot_name,
        DBInstanceIdentifier=db_identifier
    )
    
    print(f"Created snapshot: {snapshot_name}")

def cleanup_old_snapshots():
    """清理 30 天前的快照"""
    cutoff_date = datetime.now() - timedelta(days=30)
    
    snapshots = rds.describe_db_snapshots(
        DBInstanceIdentifier='my-production-db'
    )
    
    for snapshot in snapshots['DBSnapshots']:
        if snapshot['SnapshotCreateTime'] < cutoff_date:
            rds.delete_db_snapshot(
                DBSnapshotIdentifier=snapshot['DBSnapshotIdentifier']
            )
            print(f"Deleted old snapshot: {snapshot['DBSnapshotIdentifier']}")
```
