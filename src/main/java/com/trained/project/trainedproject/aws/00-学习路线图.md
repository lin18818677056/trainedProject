## 🎯 学习建议

### 学习顺序
1. 按顺序阅读每个模块，不要跳读
2. 每个章节都要动手实践
3. 完成每章的练习题
4. 最后完成实战项目

### 学习时间规划
- **第 1-2 天**: 基础概念和账户设置 (01-02 章)
- **第 3-5 天**: 核心服务学习 EC2、S3 (03-04 章)
- **第 6-8 天**: 进阶服务 RDS、VPC (05-06 章)
- **第 9-10 天**: 无服务器和权限管理 (07-08 章)
- **第 11-15 天**: 实战项目练习 (09 章)
- **持续学习**: 最佳实践和深入探索 (10 章)

### 实践环境准备
- 一台可以上网的电脑
- 信用卡或借记卡（用于 AWS 账户验证）
- 每天至少 2 小时学习时间
- 准备好做笔记

### ⚠️ 重要提醒

#### 费用控制
1. **AWS 免费套餐**: 新账户有 12 个月免费套餐
2. **设置预算告警**: 在 Billing Dashboard 设置每月预算
3. **及时关闭资源**: 不用的 EC2 实例要及时停止/终止
4. **关注用量**: 定期检查账单和使用情况

#### 安全注意
1. **保护根账户**: 启用 MFA 多因素认证
2. **使用 IAM 用户**: 日常操作使用 IAM 用户而非根账户
3. **最小权限原则**: 只授予必要的权限
4. **定期轮换密钥**: Access Key 要定期更换
